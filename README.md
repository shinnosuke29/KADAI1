# plus コマンド

`plus` コマンドは、標準入力から受け取った数値の合計を計算するシンプルなツールです。

## 使い方

以下のように標準入力を渡して、合計を取得できます。

```bash
$ echo -e "1\n2\n3\n4\n5" | python3 scripts/plus.py
15
